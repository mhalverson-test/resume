Other Projects

Equine Lameness Detection System (ELDS)
Jan 2012 - June 2012
  • In a team of four students, created a device for our client that measures the weight distribution across all four hooves of a horse. Starting with basic concept generation, the team worked through the entire design process, ultimately manufacturing and testing a successful prototype. The device design included mechanical calculations, circuit design, wireless data transmission, and custom software. The process involved rigorous documentation, reporting, budgeting, material sourcing, machining, soldering, packaging, and coding.
  • The software was a Python GUI tool for managing the collection of data (4 concurrent input streams: one per hoof), visualizing the live-streaming data feeds from each hoof, and performing data analysis to calculate the number of weight shifts on each hoof in a given time period.
  • Sole developer of the Python GUI tool.
